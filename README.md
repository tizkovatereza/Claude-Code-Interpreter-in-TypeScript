# Claude-Code-Interpreter-in-TypeScript


### Techstack
- E2B Code Interpreter SDK
- Claude by Anthropic
- JavaScript/TypeScript


### How to run the program
- Prerequisites
  - E2B API KEY
  - ANTHROPIC API KEY

- Install packages
  

```
npm init -y \
&& npm i --save-dev typescript tsx @types/node \
&& npm i @e2b/code-interpreter
```


```
npm install @anthropic-ai/sdk
```
